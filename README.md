# 项目简介 java561

本项目是一套基于SpringBoot的简单通用后台管理系统，包含了源代码及数据库文件。

## 技术栈

- **前端**：layui
- **后端**：SpringBoot + Thymeleaf + Spring Security + MySQL

## 功能模块

- 权限管理
- 菜单管理
- 用户管理
- 系统设置
- 实时日志
- 实时监控
- API加密
- 登录用户修改密码
- 配置个性菜单

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 开发环境：IDEA

## 部署步骤

1. 准备所需的环境，包括Java、MySQL等。
2. 导入项目源代码至IDEA。
3. 配置数据库连接信息。
4. 运行项目。

## 目录结构

```
project-root/
│
├── src/
│   ├── main/
│   │   ├── java/
│   │   ├── resources/
│   │   └── webapp/
│   └── test/
│
└── db/
    └── schema.sql
```

## 核心亮点

- 系统基于SpringBoot，易于上手和维护。
- 集成了Spring Security，为系统提供了可靠的安全保障。
- 提供了实时日志和监控功能，方便系统运行状态的及时了解。
- 支持API加密，提高了数据传输的安全性。
- 支持个性化配置，可满足不同用户的需求。
